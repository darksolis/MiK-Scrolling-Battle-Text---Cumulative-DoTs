MSBT CUMULATIVE DOT TRACKER - DARKSOLIS PREMIUM v5.4.84

PURPOSE
Replaces a stream of individual periodic-damage ticks with a fixed cumulative total positioned at an existing MSBT scroll area. Normal MSBT damage continues to animate normally around it.

OPEN SETTINGS
/msbt
Select the Cumulative DoTs tab. /msbtdot remains a shortcut to the same interface.

RESET MODES
Target death: each target accumulates until its death event, optional combat-end clear, or the configurable out-of-combat fallback.
DoT expires: each spell application resets when refreshed/reapplied and is removed when that application ends.

DISPLAY MODES
Separate totals: one neatly spaced sticky row per target.
Combined total: one number combining all tracked targets.

PREMIUM DISPLAY CONTROLS
- Inherit the selected MSBT scroll area's font or choose any registered MSBT font.
- Adjustable font size, text color, outline, alignment, row spacing, and X/Y offsets.
- Target names toggle.
- Spell names toggle.
- Per-spell damage breakdown toggle.
- Spell icons toggle.
- Active-only spell labels toggle.
- Exact or abbreviated totals.
- Critical periodic-tick pulse.
- Maximum target rows and spell labels.
- Sort by recent damage, highest cumulative damage, or target name.

TRACKING FILTERS
Track all periodic damage or disable that setting and build a spell whitelist using exact spell names or spell IDs. Pet periodic damage is optional.

TESTING
Use Test Display in /msbt > Cumulative DoTs. The sample rows update live as settings change.
